JavaScript

发表于 2016-12-25 | 分类于 JavaScript |

虽然很久之前就接触过JavaScript，陆陆续续也一直有在用JavaScript，但是还是对JavaScript懵懵懂懂，仿佛处于云山雾里，对整个JavaScript没有一个系统的了解，所以趁这段时间空闲时间比较多，准备系统学习一下JavaScript，顺便把学到的一些点记录下来。

好多人都推荐Nicholas C.Zakas写的《JavaScript高级程序设计》（曹力 等译）这本书，我读了一下发现还不错，强烈推荐大家都读一下。这本书现在已经出到第三版了，但是相比飞速发展的前端来说还是有点陈旧了，对里面没有涉及到的ES6以后再单独学习一下。

JavaScript简介

历史

1995年，Brendan Eich发明了LiveScript，随后为了利用Java的影响力，Netscape将LiveScript改名为JavaScript。随后微软也推出了JScript，再加上CEnvi推出的ScriptEase，早期的JavaScript三足鼎立，但是没有标准，非常混乱。这种情况非常不利于web的发展，后来ECMA协调各大公司，制定了JavaScript的标准ECMAScript 1，后来各家JavaScript都慢慢实现了ECMAScript的标准。2015年6月17日，ECMAScript 2015（即ES6，以后的官方都采用ECMAScript+年份命名）标准正式发布了。

JavaScript

JavaScript是什么？JavaScript跟ECMAScript不是一个东西，JavaScript包含了：

  • ECMAScript
  • DOM 文档对象模型 DOM让我们能跟Web内容进行交互
  • BOM 浏览器对象模型 BOM让我们能够跟跟浏览器进行交互
